## Upcoming updates

#### <span className='text-yellow-200'>Styling and class styling for subcomponents of `Quiz` instead of just the parent container.</span>
- `Options` are displayed in a straight column rather than 2x2.
- `Question` is displayed in a different font size and color.

#### <span className='text-yellow-200'>New quiz formats</span>
- `Quiz` will in the future support only test formats that have correct answers.
- `Questionnaire` will be introduced to support formats that do not have correct answers, e.g. personality tests, matrix.

#### <span className='text-yellow-200'>Templates (starter-kit)</span>

Create a `true-false` quiz using `single-answer` format

## Further down the road
- Callback function prop that accepts hook to allow retrieving of data (currentScore, results) from Quiz.
- Analysis report in results page, and custom results page.
- Support mixed types in question and options, e.g. an image as question or a mix of image and text as options.
